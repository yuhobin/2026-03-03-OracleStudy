# 📅 2026-04-29 MiniProject_day6
### 🏷️ Topic
도서관리 프로젝트 회원가입 페이지 생성

### 📝 What I Learned
프로젝트에서 내가 담당한 회원가입 페이지를 만들었다. 로그인과 같은 Member 테이블 사용하고 추가로 우편주소 검색을 위한 Zipcode 테이블을 생성 후 사용

진행 순서 

          먼저 메커니즘 파악
          
          공용DB에 우편 주소 데이터 수집하여 Zipcode 테이블 생성 후 데이터 추가
          
          회원가입 프레임 만들기(윈도우 창으로 생성)
          
          아이디 중복여부, 우편주소 검색 가능에 중점

          

DAO 제작 공통사항 

                  1. 멤버 변수 및 DBUtil 객체 생성
                  
                  2. 싱글턴 패턴
                  
                  3. 오라클 연결 (DBUtil 활용)
                  
                  4. 오라클 해제 (DBUtil 활용)

회원가입 기능 구현

                아이디 존재여부 확인  
                String sql="SELECT COUNT(*) "
					                  +"FROM member "
					                  +"WHERE memid=?";

                사용자 정보 일치여부 확인
                String sql="SELECT pwd,memid,isadmin "
