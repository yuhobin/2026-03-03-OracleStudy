# 📅 2026-04-28 MiniProject_day5
### 🏷️ Topic
도서관리 프로젝트 로그인 페이지 생성

### 📝 What I Learned
프로젝트에서 내가 담당한 로그인 페이지를 만들었다

진행 순서 

          먼저 메커니즘 파악
          
          공용DB에 회원 정보 수집하여 추가
          
          관리자 여부 설정
          
          로그인창 프레임 및 버튼 만들기(윈도우 창으로 생성)
          
          MemberVO를 만들어 데이터의 컬럼을 받을 변수 설정, MemberDAO를 만들어 자바와 오라클을 연동(이번 프로젝트의 핵심은 DAO 제작 => 웹에서도 똑같이 적용되기 때문), 
          메인 홈 버튼을 만들 Menupanel 제작(어떤 페이지를 만들지 구상), 화면 이동을 통제할 ControllerPanel 제작, 
          눈에 보이는 동작을 담당 할 UserMainForm 제작

DAO 제작 공통사항 

                  1. 멤버 변수 및 DBUtil 객체 생성
                  
                  2. 싱글턴 패턴
                  
                  3. 오라클 연결 (DBUtil 활용)
                  
                  4. 오라클 해제 (DBUtil 활용)

로그인 기능 구현

                아이디 존재여부 확인  
                String sql="SELECT COUNT(*) "
					                  +"FROM member "
					                  +"WHERE memid=?";

                사용자 정보 일치여부 확인
                String sql="SELECT pwd,memid,isadmin "
                          +"FROM member "
                          +"WHERE memid=?";
