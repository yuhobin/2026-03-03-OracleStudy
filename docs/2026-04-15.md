내장함수의 단일행 함수, GROUP BY 학습

### 📝 What I Learned
내장함수 중 단일행 함수에  대해 학습했다.

1. 문자 함수 :

              ***LENGTH / LENGTHB => 문자의 개수 / 문자의 바이트 수

              LPAD(문자열, 글자수, '변경할 문자')
		  		    LPAD('KING',5,'#') ==> #KING
			      
			      	RPAD(문자열, 글자수, '변경할 문자')
			      	RPAD('KING',5,'#') ==> KING#

              UPPER(문자열) => 대문자 변경
              LOWER(문자열) => 소문자 변경
              INITCAP(문자열) => 첫자만 대문자

               ***REPLACE : 변경 ===> (크롤링 시==> & => ^)
               REPLACE(문자열, 찾는 문자, 변경할 문자)

			          TRIM : 공백 / 특정 문자 제거 (자바 => 공백만 제거)
                LTRIM('문자열'), LTRIM('문자열', '제거할 문자') => 왼쪽 공백 제거
                RTRIM('문자열'), RTRIM('문자열', '제거할 문자') => 오른쪽 공백 제거 
                TRIM : 좌우 공백 제거  TRIM (문자 FROM 문자열)  => 좌우 공백 제거

                ASCII : 문자 => 숫자변환  / CHR : 숫자 => 문자변환

                ******SUBSTR(문자열, 시작위치, 개수) => 문자번호 => 1번부터 시작

                INSTR(문자열, 찾을문자, 시작위치, 몇번째) => indexOf (양수), lastIndexOf(음수)

                 CONCAT : 문자열 결합 => ||
			           CONCAT('문자열', '결합될 문자열')
   
2. 숫자 함수 :
   
                 MOD : % 나머지  => MOD(10, 2) 10%2

4. 변환 함수 :  
   
                TO_CHAR : 숫자, 날짜 => 문자열 반환
          			TO_NUMBER : 문자 => 숫자
			          TO_DATE : 문자 => 날짜
